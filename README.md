# Balasooriya Pvt Hospital — Website

## Overview
The public-facing website for Balasooriya Pvt Hospital.

## Pages
- `index.html` — Home
- `about.html` — About Us
- `services.html` — Services
- `doctors.html` — Doctors
- `contact.html` — Contact (form + hospital details)

## Tech Stack
- HTML5 / CSS3 / vanilla JS
- May align with the Node/Express + MySQL stack used on the CBRMS project for
  consistency, to be confirmed once requirements are finalized.

## Current State
Static multi-page site with placeholder content (doctor names, address, phone, etc.)
ready to be swapped for real hospital details. Contact form is front-end only — no
backend/email integration yet.

## Next Steps
- Replace placeholder content (doctors, address, phone, services) with real details
- Connect the contact form to a backend or email service
- Design the appointment booking flow
- Decide hosting/deployment approach
